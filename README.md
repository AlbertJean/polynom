# react-webpack-boiler
This is a simple React project boilerplate with webpack and redux.

This boilerplate includes:

- react
- react-dom
- react-transform-hmr
- react-redux
- redux
- webpack
- webpack-dev-server
- babel
- postcss
- eslint
- stylelint

## Start the dev server

```
$ npm start
```

## Build all files

```
$ npm run prod
```
